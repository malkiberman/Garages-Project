פרויקט מוסכים 
לאפליקציה - https://garages-project.onrender.com

להרצה בסביבת פיתוח:
עליך לודא שמותקנים אצלך
.NET SDK
Angular CLI

בפרויקט .NET להריץ את הפקודות:

dotnet restore - לשחזר את התלויות
dotnet run להרצת הפרויקט

בפרויקט  Angular 
npm install להתקין תלויות
ng serve להריץ אפליקציה

והאפליקציה רצה בכתובת
http://localhost:4200
